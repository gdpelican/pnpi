== README
{<img src="https://codeclimate.com/github/gdpelican/pnpi.png" />}[https://codeclimate.com/github/gdpelican/pnpi]

PNPI is a database application for the Philadelphia New Play Initiative,
designed to allow theater artists to easily connect with one another.

It's written in Ruby 2.0, Rails 4.0, and Knockout.

You can view it online here[http://pnpi.herokuapp.com]

Any issues feel free to contact james.kiesel@gmail.com[mailto:james.kiesel@gmail.com]
